### Parse Srt File Using JianYing

#### Useage
- Fork this Repo
- Enable actions
- Edit Config.json
```json
{
    "types":"",     *"requests","you-get","bili"*
    "ASDB":true,
    "url":["BVXXXXXX","https://a.b.c"]
}
```
- It will Pubished to github release
